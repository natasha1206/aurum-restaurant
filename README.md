Aurum Restaurant Website
========================

ğŸš€ Getting Started
------------------

### Prerequisites

Make sure you have one of the following installed:

*   **Node.js** (v18+ recommended)
    
*   **Bun** (optional, but supported)
    

### Installation

Using npm:

`   npm install   `

Using Bun:

`   bun install   `

### Development Server

`   npm run dev   `

or

`   bun run dev   `

The site will be available at:

`   http://localhost:5173   `

### Production Build

`   npm run build   `

or

`   bun run build   `

Preview the production build locally:

`   npm run preview   `

ğŸŒ Deployment
-------------

The project is configured for **GitHub Pages** deployment.

Key points:

*   Uses **Vite** with a custom base path

*   Deployment is handled via the gh-pages branch
    

Deploy with:

`   npm run deploy   `
